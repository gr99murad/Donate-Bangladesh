Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a hierarchical representation of a webpage that allows scripts to dynamically access and manipulate HTML or XML content.

How do you select an element from the DOM?

ways to select an element 1.By ID ---- document.getElementById('id')
2.By class ---document.getElementByClassName('class')
3.By tag -----document.getElementByTagName('tag')
4.css selector-- document.querySelector('selector') ---document.querySelectorAll('selector')

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener on a parent element manages events for its child elements using event bubbling. This improves performance by reducing the number of listeners and simplifies handling dynamically added elements.It checks the event's target to determine which child triggered the event

How do you manipulate an elements attributes and styles using the DOM?

I can manipulate an element's attributes using element.setAttribute('attribute', 'value') and retrieve them with element.getAttribute('attribute'). For styles, use element.style.property = 'value' to set a style or element.style to access multiple styles at once.
